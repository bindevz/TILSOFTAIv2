# SQL Module Template (Starter)

Purpose: copy this folder to create a new module without changing core code.

Contents
- `001_views_template.sql` - stub semantic view(s)
- `002_sps_ai_template.sql` - stub ai_* stored procedure(s)

Notes
- These scripts are templates only; do not deploy in production as-is.
- Replace `<module>` naming and update view/SP names.
- All ai_* SPs must return the standard JSON envelope.
